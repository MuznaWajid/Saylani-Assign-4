Introduction to MS Office

Microsoft Office, is more commonly called as MS Office. It was first introduced in 1988 by Bill Gates. Basically it is an office suite of various applications, servers and services. It includes applications, servers and services developed by Microsoft. In the initial version there were only Microsoft Word, Excel and PowerPoint. Later it has grown substantially with many shared features such as common spell checker, OLE data integration and visual basic for application scripting language. It also functions as a development platform for line of business software. Different versions of Microsoft office have been introduced keeping in mind the different users and working environments. The most popular and widely used one among them is the desktop version. It is available for computers running on Windows and Mac OS operating systems. Apart from this Microsoft has a Mobile Office, free-to-use versions of office applications for mobile phones and Office online a web based version of office apps.
Introduction to MS Word
It is a word processor included in the Microsoft Office suite. MS Word has been included in the very first version of MS Office and it is available even in the latest version. It was a part of the now discontinued MS Works too. Actually, the first version of MS Word was released even earlier to the release of MS Office. Word was released in 1983 for the MS Disk operating system. Then Word for Mac was released in 1985 and this was the first graphical version of MS Word. In the initial stages Word was an implementing proprietary doc. Format as its primary format. However this format was abandoned in favor of Open Office XML format. Portable Document Format or PDF and Open Document Format or ODF have been introduced for the first time in Windows 2007 with service pack 2.
Introduction to Excel
Microsoft Excel or MS Excel is a spreadsheet and the first version of Excel for Mac OS was released in 1985. The second version for Windows (Version 2.0) was released in 1987.
Introduction to Powerpoint
This is a presentation program used to create slides with images and many other objects. These slides can be displayed on screens or can be printed on transparencies. The above mentioned three were available in MS Office when it was introduced for the first time. Later on many additions have been made and they are as follows.
Microsoft Access:
– A database management system for Windows
Microsoft Outlook:
– A personal information manager
Microsoft OneNote:
– A note taking program that takes handwriting and types notes.
Microsoft Publisher:
– A desktop publishing app
Microsoft Project:
– A project management app
Microsoft Visio:
– A diagram and flowcharting app
